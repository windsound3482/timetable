GTFS EDITOR

Instruction for users

GTFS Editor is a powerful webbased Editor, which provides you many efficient tools to adjust the gtfs files which is already exist, as well as create your own GTFS files.

The General Transit Feed Specification (GTFS) defines a common format for public transportation schedules and associated geographic information. You can find the detailed reference here: Google GTFS Format

Press download button to get an example for the GTFS Feed.

Join us for developers

Timetable Project(for GTFS Editor) is an opensource project, which was founded at the Technical University of Darmstadt. The Project is coded unter Typescript+Angular. Please join us to contribute your contribution to the open-source softwares. We believe "Community over Code".
